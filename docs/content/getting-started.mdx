Superkey is a stylized command palette component built with React, Tailwind CSS & HeadlessUI ready to be used in your next project.

### 1. Install:

ðŸš§ Superkey only works with React 18 or higher. It also requires [headlessUI](https://headlessui.com/) to be installed.

```bash
npm install superkey @headlessui/react
# or
yarn add superkey @headlessui/react
# or 
pnpm install superkey @headlessui/react
```

### 2. Import styles:

```ts
import "superkey/styles.css";
```

### 3. How to use:

```jsx {5-19}
import { Command, CommandInput, CommandList, CommandOption } from "superkey";

function App() {
  return (
    <Command open={true}>
      <CommandInput onChange={(e) => {
        console.log(e.target.value);
      }} />
      <CommandList>
        <CommandOption value="Option 1">
          <h1>Option 1</h1>
          <p>Description</p>
        </CommandOption>
        <CommandOption value="Option 2">
          <h1>Option 2</h1>
          <p>Description</p>
        </CommandOption>
      </CommandList>
    </Command>
  );
}
```
