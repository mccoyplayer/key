---
title: 🎨 Default colors
description: Default colors of each component.
---

## 🛠️ Command

### Overlay:

- Use: `overlayClassName` to edit the styles.

```txt
bg-gray-200/60 dark:bg-zinc-900/60
```

### All component:

- Use: `className` to edit the styles.

```txt
bg-white dark:bg-zinc-900 dark:ring-zinc-900/5 border border-neutral-300 dark:border-neutral-800
```

## 🛠️ CommandInput

### Input:

- Use: `inputClassName` to edit the styles.

```txt
dark:text-neutral-200
```

## 🛠️ CommandList

- No default colors 🤔.

## 🛠️ CommandOption

- Use `activeClassName` to edit the styles.

```txt
Active: No default colors.
No active: bg-transparent
```
